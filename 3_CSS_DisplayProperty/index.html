CSS Display Properties Examples

display: block
Block Element 1
Block Element 2
<div class="block-example">Block Element 1</div>
<div class="block-example">Block Element 2</div>

.block-example {
  display: block;
  background: lightblue;
  margin-bottom: 10px;
  padding: 10px;
}

Notes:

  • Element takes up the full width available (its container).
  • Causes a line break before and after the element.
  • Used for block-level elements like <div>, <p> etc.
display: inline
Inline 1 Inline 2 
<span class="inline-example">Inline 1</span>
<span class="inline-example">Inline 2</span>

.inline-example {
  display: inline;
  background: yellow;
  padding: 5px;
  margin-right: 5px;
}

Notes:

  • Element only takes up as much width as necessary.
  • Does not cause a line break.
  • Elements sit next to each other on the same line.
display: inline-block
Box 1
Box 2
<div class="inline-block-example">Box 1</div>
<div class="inline-block-example">Box 2</div>

.inline-block-example {
  display: inline-block;
  width: 100px;
  height: 50px;
  background: pink;
  margin-right: 10px;
  text-align: center;
  line-height: 50px;
}

Notes:

  • Similar to inline but allows setting width and height.
  • Elements appear on the same line but can have dimensions.
  • Useful for layouts like buttons, cards, etc.
display: flex
Flex 1
Flex 2
Flex 3
<div class="flex-container">
  <div class="flex-item red">Flex 1</div>
  <div class="flex-item blue">Flex 2</div>
  <div class="flex-item green">Flex 3</div>
</div>

.flex-container {
  display: flex;
  gap: 10px;
  background: lightgray;
  padding: 10px;
}

.flex-item {
  width: 100px;
  height: 50px;
  text-align: center;
  line-height: 50px;
  color: white;
}

Notes:

  • Creates a flexible container for its items.
  • Items can be aligned using properties like justify-content and align-items.
  • Great for layouts that require flexible positioning of items.
display: grid
Grid 1
Grid 2
Grid 3
<div class="grid-container">
  <div class="grid-item purple">Grid 1</div>
  <div class="grid-item cyan">Grid 2</div>
  <div class="grid-item magenta">Grid 3</div>
</div>

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
  background: lightyellow;
  padding: 10px;
}

.grid-item {
  height: 50px;
  text-align: center;
  line-height: 50px;
  color: white;
}

Notes:

  • Creates a grid layout with rows and columns.
  • Define grid structure using grid-template-columns and grid-template-rows.
  • Items can span multiple rows or columns using grid-row and grid-column.
  • Best for precise 2D layouts.
display: table
Cell 1
Cell 2
<div class="table-container">
  <div class="table-row">
    <div class="table-cell">Cell 1</div>
    <div class="table-cell">Cell 2</div>
  </div>
</div>

.table-container {
  display: table;
  border: 1px solid black;
}

.table-row {
  display: table-row;
}

.table-cell {
  display: table-cell;
  padding: 10px;
  border: 1px solid black;
}

Notes:

  • Mimics the behavior of HTML elements.
  • Use table-row for rows and table-cell for individual cells.
  • Great for tabular data without semantic HTML.

    display: inline-grid
    A
    B
    <div class="inline-grid-container">
  <div class="inline-grid-item orange">A</div>
  <div class="inline-grid-item teal">B</div>
</div>

.inline-grid-container {
  display: inline-grid;
  grid-template-columns: auto auto;
  gap: 5px;
  background: lavender;
  padding: 10px;
}

.inline-grid-item {
  padding: 10px;
}

    Notes:

    • Works like grid but remains inline within its parent.
    • Aligns items using grid properties like grid-template-columns.
    • Useful for small grid layouts that need to remain inline.
