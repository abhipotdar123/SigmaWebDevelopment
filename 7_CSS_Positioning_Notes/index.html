Understanding CSS Position Property

The CSS position property specifies how an element is positioned in the document. Here are the common values:

1. static (Default)

Elements are positioned according to the normal flow of the document. No special positioning applied.

Static
.static {
  background-color: #3498db;
  position: static;
}

2. relative

The element is positioned relative to its normal position. You can use top, right, bottom, and left to offset it.

Relative
.relative {
  background-color: #e74c3c;
  position: relative;
  top: 10px;
  left: 20px;
}

3. absolute

The element is positioned relative to its nearest positioned ancestor (other than static). If no such ancestor exists, it is positioned relative to the html element.

Absolute
.absolute {
  background-color: #2ecc71;
  position: absolute;
  top: 50px;
  left: 50px;
}

4. fixed

The element is positioned relative to the viewport and does not move when the page is scrolled.

Fixed
.fixed {
  background-color: #9b59b6;
  position: fixed;
  top: 20px;
  right: 20px;
}

5. sticky

The element toggles between relative and fixed depending on the scroll position. It remains relative until it crosses a defined threshold (top, left, etc.).

Sticky
.sticky {
  background-color: #f1c40f;
  position: -webkit-sticky; /* For Safari */
  position: sticky;
  top: 0;
}
